Правила

  1. Для участия в Соревнованиях командам-участницам необходимо пройти регистрацию на сайте Соревнований, после чего подключиться к игровому серверу в указанное на сайте время.
  2. Команда состоит не более чем из 10 участников, среди которых обязательно присутствует капитан;
  3. Состав команды нельзя менять после начала соревнований;
  4. Не позднее, чем за 48 часов до начала соревнований, командам будут отправлены учетные данные для доступа к странице с тасками. Учетные данные будут высланы на адрес капитана, указанный при регистрации;
  5. Соревнования длятся сорок восемь часов;
  6. На Соревнованиях каждой команде предлагается выполнить задания Соревнований;
  7. Во время проведения Соревнований участникам необходимо набрать как можно больше баллов;
  8. Для выполнения заданий командам-участницам в 2012 году будет предложен архив заданий, распределённых по некоторым из перечисленных ниже категориям:
    • Crypto –криптографические задачи;
    • Web – задачи, связанные с разнообразными веб-уязвимостями;
    • StegaSic (Steganography & Forensic) – стеганография и исследования в области компьютерных преступлений;
    • PPC (Professional programming and coding) – различные задачи на программирование;
    • Admin – задачи на администрирование сетей;
    • Joy – задачи, нацеленные на общую эрудицию участников в сфере информационной безопасности, а также задания развлекательного характера.
  9. Каждая категория состоит из 5 заданий, ранжированных по сложности. Выполнение задания из каждой категории приносит команде указанное возле названия задания количество баллов (от 100 до 500). Изначально для решения в каждой категории доступно только 1 задание, следующие становится доступным только после решения какой-либо командой последнего из открытых заданий в данной категории;
  10. Жюри Соревнований оставляет за собой право открывать задания по своему усмотрению в любой момент Соревнований, информируя об этом команды в новостях на сайте за 5 минут до открытия;
  11. В результате успешного выполнения задания, команда получает кодовую строку (флаг), которую она должна вписать в онлайн-форму, предоставленную на сервере Соревнований. Флаг представляет собой осмысленную фразу, по тематике близкую к тематике Соревнований;
  12. Проверка результатов выполнения заданий производится во время проведения соревнований. По результатам проверки техническая комиссия составляет таблицу результатов Соревнований;
  13. Все задания Соревнований, а также систему проверки результатов выполнения заданий разрабатывает техническая комиссия и до начала Соревнований держит их в тайне;
  14. Во время проведения Соревнований командам разрешается использовать любое количество компьютеров и сетевого оборудования;
  15. При выполнении заданий участникам Соревнований запрещается выполнять следующие действия:
    • проводить атаки на серверы жюри и проверяющую систему;
    • мешать всевозможными способами получению флага другим командами;
    • генерировать неоправданно большой объем трафика.
  16. В процессе Соревнований жюри будет публиковать новости и подсказки к заданиям. Единственное официальное место для просмотра новостей – сайт Соревнований;
  17. Во время проведения соревнований будет функционировать общая для всех участников соревнований конференция, также участникам будут выданы контакты технической комиссии для оперативного решения возможных проблем;
  18. Следить за соревнованиями можно на публичном мониторе;
  19. Жюри Соревнований оставляет за собой право вносить любые изменения в данный технический регламент до начала соревнований;
  20. За нарушение требований настоящего Технического регламента или нарушение хода соревнований другим способом команда может быть оштрафована или дисквалифицирована по решению жюри;
  21. За обнаружение уязвимостей в системах и сервисах организаторов Соревнований а также за предложения по их устранению команда может быть поощрена по решению Жюри.

Исключительные ситуации

К возможности возникновения исключительных ситуаций команды должны отнестись с пониманием. Организаторы приложат все возможные усилия для устранения таких ситуаций.
